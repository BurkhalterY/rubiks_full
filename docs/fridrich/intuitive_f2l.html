Indroduction

Les F2L (First 2 Layers) sont une méthode permettant de résoudre les deux premières couronnes en même temps. Elle consiste à créer une paire pour ensuite la glisser dans son slot.

Définitions

Paire : 2 cubes correctement placés l'un par rapport à l'autre.
Slot : Emplacement où ira se placer la paire.

Type 1: Cas de base

Un cas de base peut être résolu en 3 mouvements.

« Ce n’est pas la paire qui va vers le slot, c’est le slot qui va vers la paire. »
